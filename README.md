# Ravi Kovind | Portfolio

Personal portfolio website showcasing my work as a Software Engineer.

**Live:** [ravikovind.github.io](https://ravikovind.github.io)

![Portfolio Screenshot](assets/ss.png)

## Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
- Canvas API (Particle Animation)
- Responsive Design
- GitHub Pages

## Features

- Interactive particle network animation
- Responsive mobile-first design
- Smooth scroll animations
- Glass-morphism UI elements
- SEO optimized with structured data

## Sections

- **Hero** - Introduction with particle animation
- **About** - Background and engineering philosophy
- **Experience** - Professional work history
- **Projects** - Featured projects (TingTing, MCPVave, Tredye)
- **Tech Stack** - Skills and technologies
- **Contact** - Social links and email
